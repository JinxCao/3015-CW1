This is a 3D graphics rendering project based on OpenGL, demonstrating the application of various modern graphics technologies, including lighting models, texture mapping, normal mapping, skyboxes, post-processing effects, and interactive camera control.

Drag the mouse to rotate the view.
Use the scroll wheel to control the zoom distance.
Number key 1: Toggle the directional light switch
Number key 2: Toggle the spotlight switch
Number key 3: Toggle the display of the sphere/model
Number key 4: Toggle the auto-rotation animation
Letter key F: Toggle the post-processing effect (grayscale)
ESC key: Exit the program
Spacebar: Pause/Resume the animation.

Use release.
